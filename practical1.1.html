About HTML

HTML stands for HyperText Markup Language. It is the standard markup language for creating web pages. HTML describes the structure of a web page using markup tags.

Text Formatting Examples

Here are some common text formatting tags:

  • Bold text uses the <strong> tag.
  • Italic text uses the <em> tag.
  • Underlined text uses the <u> tag.
  • Highlighted text uses the <mark> tag.
